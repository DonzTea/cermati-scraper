# Cermati Scraper

Scraping tools for [https://www.cermati.com/artikel](https://www.cermati.com/artikel).

## How to run

> 1. install Node.js
> 2. clone repository
> 3. execute `npm install`
> 4. execute `npm audit fix`
> 5. execute `node solution.js`
> 6. now scraping result saved on `solution.json` file
